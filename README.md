# YZ SMART FACTORY

## Descrição
Este repositório contém o código-fonte do site/sistema da **Metalúrgica YZ**. Trata-se de uma aplicação web full-stack desenvolvida com tecnologias modernas para gerenciamento e apresentação dos serviços da metalúrgica.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

### Front-end
* **TypeScript** (Principal linguagem do projeto)
* **Vite** (Ferramenta de build)
* **HTML5 / CSS3**

### Back-end & Banco de Dados
* **Node.js** (`server.ts` - servidor da aplicação)
* **Python** (`app.py` - rotinas/scripts em Python)
* **Supabase** (Banco de dados relacional e Autenticação)

## 📁 Estrutura do Projeto

Abaixo estão os principais arquivos e pastas do projeto:

* `/src` - Código-fonte principal da aplicação (componentes, páginas, estilos).
* `/public` - Arquivos estáticos e assets públicos.
* `server.ts` - Arquivo principal do servidor Node.js.
* `app.py` - Arquivo auxiliar com implementações em Python.
* `supabase-schema.sql` - Arquivo de configuração e estruturação do banco de dados no Supabase.
* `vite.config.ts` - Arquivo de configuração do empacotador Vite.
* `requirements.txt` - Dependências do ecossistema Python.
* `package.json` - Dependências do ecossistema Node.js.
